Chat trigger
~~~~~~~~~~~~

Use the :doc:`n8n Chat Trigger <../integrations/builtin/core-nodes/n8n-nodes-langchain.chattrigger/index>` to trigger a workflow based on chat interactions.

Chatbot widget
~~~~~~~~~~~~~~

n8n provides a chatbot widget that you can use as a frontend for AI-powered chat workflows. Refer to the `@n8n/chat npm page <https://www.npmjs.com/package/@n8n/chat>`_ for usage information.
