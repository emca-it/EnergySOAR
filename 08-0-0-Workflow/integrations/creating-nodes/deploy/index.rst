Deploy a node
==============

This section contains details on how to deploy and share your node.

You can choose to:

* `Submit your node to the community node repository </08-0-0-Workflow/integrations/creating-nodes/deploy/submit-community-nodes.html>`_. This makes it available for everyone to use, and allows you to `install and use it </08-0-0-Workflow/integrations/community-nodes/installation/index.html>`_ like any other community node. This is the only way to use custom nodes on cloud.
* Install the node into your n8n instance as a `private node </08-0-0-Workflow/integrations/creating-nodes/deploy/install-private-nodes.html>`_.
