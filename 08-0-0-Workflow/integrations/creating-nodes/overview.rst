Creating nodes
==============

Learn how to build your own custom nodes.

This section includes:

* Guidance on planning your build, including :doc:`which style to use <plan/choose-node-method>`.
* :doc:`Tutorials <build/index>` for different node building styles.
* Instructions for :doc:`testing your node <test/index>`, including how to use the n8n :doc:`node linter <test/node-linter>` and :doc:`troubleshooting <test/troubleshooting-node-development>` support.
* How to :doc:`share your node <deploy/submit-community-nodes>` with the community, submit it for :doc:`verification by n8n <deploy/submit-community-nodes>`, or use it as a :doc:`private node <deploy/install-private-nodes>`.
* :doc:`Reference material <build/reference/index>`, including UI elements and information on the individual files that make up a node.

Prerequisites
^^^^^^^^^^^^^

This section assumes the following:

* Some familiarity with JavaScript and TypeScript.
* Ability to manage your own development environment, including git.
* Knowledge of npm, including creating and submitting packages.
* Familiarity with n8n, including a good understanding of `data structures </08-0-0-Workflow/data/data-structure.html>`_ and `item linking </08-0-0-Workflow/data/data-mapping/data-item-linking/index.html>`_.
