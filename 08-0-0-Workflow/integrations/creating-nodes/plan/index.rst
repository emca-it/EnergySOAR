Plan a node
===========

This section provides guidance on designing your node, including key technical decisions such as choosing your node building style.

When building a node, there are design choices you need to make before you start:

* Which `node type </08-0-0-Workflow/integrations/creating-nodes/plan/node-types.html>`_ you need to build.
* Which `node building style </08-0-0-Workflow/integrations/creating-nodes/plan/choose-node-method.html>`_ to use.
* Your `UI design and UX principles </08-0-0-Workflow/integrations/creating-nodes/plan/node-ui-design.html>`_
* Your node's `file structure </08-0-0-Workflow/integrations/creating-nodes/build/reference/node-file-structure.html>`_.
