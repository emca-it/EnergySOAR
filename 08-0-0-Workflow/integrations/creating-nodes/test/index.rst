Test a node
===========

This section contains information about testing your node.

There are two ways to test your node:

* Manually, by `running it on your own machine </08-0-0-Workflow/integrations/creating-nodes/test/run-node-locally.html>`_ within a local n8n instance.
* Automatically, using the `linter </08-0-0-Workflow/integrations/creating-nodes/test/node-linter.html>`_.

You should use both methods before publishing your node.
