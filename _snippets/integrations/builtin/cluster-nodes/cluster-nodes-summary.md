[Cluster nodes](/08-0-0-Workflow/glossary#cluster-node-n8n) are node groups that work together to provide functionality in an n8n workflow. Instead of using a single node, you use a [root node](/08-0-0-Workflow/glossary#root-node-n8n) and one or more [sub-nodes](/08-0-0-Workflow/glossary#sub-node-n8n) that extend the functionality of the node.
